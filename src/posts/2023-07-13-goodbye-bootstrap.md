> A long time ago in a galaxy far, far away....,

I am old enough to have witnessed the release of CSS 1.0 into the wild. I was still in school, but the point stands. When I started trying to make pretty websites, I was doing so with .css files.

I have used [Bootstrap](https://getbootstrap.com/) to style any and every website UI that I helped build since Bootstrap 2 was released, roughly ten years ago at this point. I honestly don't remember what I used before that. Possibly [JQuery UI](https://jqueryui.com/), since I was most likely using [JQuery](https://jquery.com/) as my de facto javascript framework at that time.

But web development has come quite a long way since then.
